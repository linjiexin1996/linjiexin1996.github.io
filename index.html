使用JS获取自己的Chrome扩展程序ID

可以通过两种不同的方式获取自己的扩展程序ID（不需要额外的权限）：

使用 runtime api
var extId = chrome.runtime.id

使用 i18n api
var extId = chrome.i18n.getMessage(“@@extension_id”)
