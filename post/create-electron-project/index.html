环境配置

  1. 安装Node.js。推荐安装最新的长期支持版本或者当前发行版本。点击访问Node.js下载页面。
  2. 安装Yarn包管理工具（Electron官方推荐）。点击访问Yarn安装页面。
  3. 安装Vue CLI。可以使用npm install -g @vue/cli或yarn global add @vue/cli命令进行安装。
